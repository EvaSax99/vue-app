# Bases TypeScript con Vue

## Descripción

Este es un proyecto diseñado para aprender y comprender las bases de TypeScript. Utiliza Vue 3 como framework de frontend junto con Vite como bundler para proporcionar un entorno de desarrollo rápido y moderno.

## Instalación de dependencias

```bash
npm install
```

## Levantar el proyecto

Para iniciar el servidor de desarrollo:

```bash
npm run dev
```

El proyecto estará disponible en `http://localhost:5173`

## Cómo estudiar

Para estudiar cada tema de TypeScript, debes ir descomentando los componentes en el archivo `src/App.vue`. 

Los componentes disponibles son:

| Componente | Tema |
|------------|------|
| `ConstLet` | Variables const y let |
| `ControlFlujo` | Estructuras de control de flujo |
| `ObjectType` | Tipos de objetos |
| `ArraysType` | Tipos de arrays |
| `BasicFunctions` | Funciones básicas |
| `DesesObj` | Desestructuración de objetos |
| `DesesArr` | Desestructuración de arrays |

**Pasos:**
1. Abre el archivo `src/App.vue`
2. Descomenta el import del componente que quieras estudiar
3. Descomenta el componente en el template
4. Guarda el archivo y visualiza los cambios en el navegador
